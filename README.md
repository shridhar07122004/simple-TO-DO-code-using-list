# 🚀 Decode Labs - Internship Tasks

This repository contains Python projects completed as part of my internship at **Decode Labs**.

---

## 📁 Project Files

### Task 1: To-Do List (`task 1 to do list.py`)
A simple **To-Do List** application built using Python's list data structure.

**Features:**
- ✅ Add tasks to the list
- 📋 View all tasks with numbering
- 🚪 Exit the program

**Concepts Used:** Lists, `append()`, `enumerate()`, loops, conditionals

---

### Task 2: Expense Tracker (`task 2 expense tracker`)
A simple **Expense Tracker** application that keeps a running total of expenses.

**Features:**
- 💰 Add expense amounts
- 📊 View total amount spent
- 🚪 Exit the program

**Concepts Used:** Variables, `float()` conversion, f-strings, loops, conditionals

---

## 🛠️ How to Run

1. Make sure Python is installed on your system.
2. Clone this repository:
   ```bash
   git clone https://github.com/shridhar07122004/internship-tasks.git
   ```
3. Navigate to the project folder:
   ```bash
   cd internship-tasks
   ```
4. Run any task:
   ```bash
   python "task 1 to do list.py"
   python "task 2 expense tracker"
   ```
